Starting fast_rsm
==================================================


#. In a new terminal enter the command

    .. code-block:: bash

        module load fast_rsm

#. If this is your first time using the fast_rsm mapper when logged into your own account, to setup a folder in your home directory enter the command

    .. code-block:: bash
        
        firstrun 

#.  Activate the fast_rsm conda environment by entering the command

    .. code-block:: bash

        activate
